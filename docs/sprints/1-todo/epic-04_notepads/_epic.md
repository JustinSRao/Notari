# Epic 04: Notepads

## Overview

Pair every flashcard with a notepad — a first-class lined-paper note editor with voice-to-text. The Notes tab provides an aggregated view across all decks so users can browse their notes independently of studying.

## Success Criteria

- [ ] Every flashcard has an associated notepad reachable from a configurable corner icon.
- [ ] Notepad UI matches _design_reference/Flashcard.html — lined paper, formatting toolbar, configurable open animation.
- [ ] User can dictate notes via voice-to-text (Apple Speech).
- [ ] Notes tab shows an aggregated index of all notepads, with deck grouping and search.
- [ ] Notepad data persists per-deck in {deck_id}_notes.json.
