# Epic 03: Flashcard Editor and Study

## Overview

Deliver the core study experience: the flashcard flip animation, the editor, the study session loop, and the optional 'both sides at once' view. This is where Notari starts feeling like a real product.

## Success Criteria

- [ ] User can create and edit flashcards (front, back, image) with a polished editor.
- [ ] Flashcards flip with a smooth 3D animation matching _design_reference/Flashcard.html.
- [ ] Flip speed, card size, and card shape are customizable.
- [ ] User can navigate through a study session and see progress.
- [ ] 'Pick up where you left off' hero card surfaces the most recent session.
- [ ] Image attached to a flashcard shows as a thumbnail; tap enlarges.
- [ ] 'Both sides visible at once' toggle replaces the flip with a stacked/side-by-side view.
