# Epic 11: Online Social

## Overview

The social layer. Usernames, friend requests, the Messages tab inbox (strictly transactional — no free-text), deck sharing/requesting, likes, friend streaks, and token gifting.

## Success Criteria

- [ ] User picks a unique username; others can search and send friend requests.
- [ ] Invite links work — tap to send a friend request to the link owner.
- [ ] Messages tab shows friend requests, deck shares, deck requests, and token gifts.
- [ ] User can share a deck with a friend; friend can preview before importing.
- [ ] User can request a deck from a friend; friend can accept and send.
- [ ] User can like a friend's deck; like count shows on the deck.
- [ ] Friend streak counts consecutive days both users hit their study goal.
- [ ] User can gift Study Tokens to a friend; recipient sees a gift item in Messages.
