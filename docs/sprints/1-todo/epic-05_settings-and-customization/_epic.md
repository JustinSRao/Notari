# Epic 05: Settings and Customization

## Overview

Expose every user-tunable setting through the Settings screen, matching the design mockup for Appearance/Accent and adding the full customization surface for flashcard, notepad, and general preferences.

## Success Criteria

- [ ] Settings screen matches _design_reference/Theme.html for Appearance + Accent sections.
- [ ] User can pick app font, flashcard customization (speed, size, shape, palette, both-sides), notepad customization (paper, line, animation, icon).
- [ ] Daily streak goal is configurable (5/10/20/50 cards).
- [ ] Notifications and Haptics toggles work and persist.
- [ ] Settings changes apply live across the app without restart.
