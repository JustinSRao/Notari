# Epic 01: Foundation

## Overview

Establish the technical foundation for Notari: a universal SwiftUI app target that runs on iOS, iPadOS, and macOS from one codebase, with a unified theme system and platform-adaptive navigation shell. Nothing user-facing yet — this epic is the runway that every later sprint takes off from.

## Success Criteria

- [ ] Universal app target builds and launches successfully on iOS Simulator, iPadOS Simulator, and macOS.
- [ ] Light/Dark/Auto appearance modes work and persist across app launches.
- [ ] All 13 accent colors are applied via a unified theme token system.
- [ ] Tab bar (iPhone) and sidebar (iPad/Mac) navigation render correctly with 5 placeholder tabs.
- [ ] CI scaffolding (lint + basic build check) runs on push.
